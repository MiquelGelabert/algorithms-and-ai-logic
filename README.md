# Algorithms and Artificial Intelligence Logic

Note: The notebooks and comments in this repository are written in Spanish.

This repository focuses on the implementation and analysis of fundamental algorithms in computer science, ranging from data sorting to artificial intelligence and graph theory.

## About this Project

This project serves as a technical deep dive into how computers solve complex problems. By implementing these algorithms from scratch, I have explored key concepts like recursion, space-time complexity, and search optimization. It represents a significant step in my journey to understand the underlying logic of efficient software development.

## Project Structure

The repository is organized into three main categories:

### 1. Artificial Intelligence and Heuristics
* **Tic-Tac-Toe Minimax:** Implementation of the Minimax algorithm. It includes Alpha-Beta pruning to optimize the decision tree, creating an unbeatable AI agent for the classic game.
* **Sudoku Solver:** A robust solver that uses the Backtracking technique to find solutions by exploring and discarding branches of possibilities efficiently.

### 2. Graph Theory and Pathfinding
* **DFS, BFS and Pathfinding:** A comparative study of Depth-First Search (DFS) and Breadth-First Search (BFS). This notebook includes a practical implementation where these algorithms are used to find and analyze routes between different towns in Mallorca, evaluating which method is more effective for navigating geographical connections.
  
### 3. Data Structures and Sorting
* **Sorting Algorithms:** A practical laboratory comparing various sorting methods. It includes performance analysis and execution time measurements to visualize the efficiency of different algorithmic approaches.

## Technical Concepts Implemented
* **Backtracking:** Systematic searching of all possible solutions.
* **Minimax Optimization:** Decision-making in zero-sum games.
* **Graph Traversal:** Logic for exploring nodes and connections.
* **Algorithm Complexity:** Observation of execution times and resource management.
